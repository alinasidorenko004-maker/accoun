Скачать БД (pgAdmin 4 v9.13 (released March 5, 2026)): 

https://www.pgadmin.org/download/pgadmin-4-windows/

Пароль от таблицы должен быть 1234 при запуске приложения, название - account

Раскрыть SERVERS
Раскрыть PostgreSQL
Databases > ПКМ > Create > databse > в поле "database" вписать account
ПКМ > account > CREATE script > вставить код из файла db.sql внутрь > нажать F5
